Pusat Bantuan

Pertanyaan paling sering ditanyakan siswa GlowMath — semoga membantu!

📝 Cara Mendaftar Kelas

  1. Buka Jelajahi Kelas di sidebar.
  2. Pilih kelas yang sesuai dengan jenjang kamu (TK/SD/SMP/SMA/UMUM).
  3. Klik Detail untuk lihat jadwal dan guru pengajar.
  4. Klik tombol Daftar — kamu akan otomatis ditambahkan ke daftar siswa kelas tersebut.

💳 Cara Bayar

Untuk saat ini pembayaran dilakukan di luar sistem — transfer manual ke rekening bimbel. Hubungi admin via WhatsApp untuk konfirmasi pembayaran. Sistem pembayaran online akan diaktifkan pada fase berikutnya.

📚 Akses Materi Kelas

Setelah kelas dimulai, materi pelajaran (PDF, link Zoom, latihan soal) bisa diakses dari halaman Kelas Saya → klik kelas → lihat tab Daftar Pertemuan. Setiap pertemuan bisa memiliki link Zoom dan file lampiran dari guru.

⭐ Beri Rating Guru & Kelas

Setelah kelas selesai (status COMPLETED), kamu bisa memberi rating bintang 1–5 untuk guru dan kelas. Banner amber otomatis muncul di halaman Kelas Saya ketika ada kelas selesai yang belum kamu rating.

📅 Cek Kehadiran & Nilai

Kehadiran kamu otomatis tercatat ketika guru menandai presensi tiap pertemuan. Lihat ringkasan di Kehadiran di sidebar. Untuk nilai quiz, tugas, UTS, UAS — buka Nilai Bulanan.

🔑 Lupa Password / Ubah Akun

Buka Pengaturan di sidebar untuk mengganti password, memperbarui nomor HP, atau update foto profil. Kalau lupa password dan tidak bisa login, hubungi admin via WhatsApp.

💬 Masih butuh bantuan?

Tim admin siap menjawab via WhatsApp dalam jam kerja (Senin–Sabtu, 08.00–20.00 WIB).
